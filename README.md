FileNameDecode
=========
### Function

It can change the file name similar to `%e6%b5%8b%e8%af%95%e6%96%87%e4%bb%b6.txt` to `测试文件.txt`

### Usage

```shell
fileNameDecode	[file(s)...]
fileNameDecode	--help
fileNameDecode	-v/--version
```

Base on [URLDecode](https://github.com/abejfehr/URLDecode)

### Install 
```shell
make
make install
```
